# 百度章鱼

    高仿的百度糯米，OC编写，采用纯代码＋xib开发，转载请注明出处，请不要用于其他商业用途。 ≖‿≖✧
    采用cocoapods管理第三方库，双击nuomi.xcworkspace启动工程，若双击nuomi.xcodeproj启动则无法正常工作。

    主要功能：
    1. 首页最新活动，附近，精品展示
    2. Web页面嵌套，JS交互
    3. 附近搜索，语音搜索 ，二维码识别
    4. 3D Touch功能
    5. 商品详细页，图片展示，商家展示
    6. 搜索记录，所在地储存（数据持久化）
    7. 通知栏展示
